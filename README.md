
# **Roulette Game**

## Required environment
	

 - Node JS version - ^v16.15.*
 - Docker

## Installation

```bash
$ npm install
```

  

## Running the app
 - Create Redis docker image
		 
```bash
 docker compose up redis -d
```
 - Run the application

```bash
$ npm run start:dev
```

## Running Redis in a browser
```bash
$ npm i -g redis-commander
```
```bash
$ redis-commander --redis -PORT
```
